Display & Positioning CSS Values

Property Values Description
z-index + (forefront) to - (background) integer values How far back or how far forward an element will appear on a web page when it overlaps other elements.
position static, relative, absolute, fixed When fixed, it is set/pinned to a specific spot on a page. The fixed element stays the same regardless of scrolling. Absolute enables an element to ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute. The absolute value removes an element entirely from the document flow. By using the positioning attributes top, left, bottom and right, an element can be positioned anywhere as expected. Relative of the CSS position property enables an element to be positioned relative to where it would have originally been on a web page.
display block, inline, inline-block Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted. Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted. Inline-block elements can appear next to each other, and can have their width and height manually adjusted.
float left, right Determines how far left or how far right an element should float within its parent element. The value left floats an element to the left side of its container and the value right floats an element to the right side of its container. For the property float, the width of the container must be specified or the element will assume the full width of its containing element.
clear left, right Specifies how an element should behave when it bumps into another element within the same containing element.The clear is usually used in combination with elements having the CSS float property. This determines on which sides floating elements are allowed to float.
